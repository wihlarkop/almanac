# Changelog

## [0.1.0] - Unreleased

### Added

- Initial `/api/v1` HTTP API for health, providers, models, validation, suggestions, catalog health, model comparison, and catalog search.
- Model catalog with provider metadata, aliases, lifecycle fields, capabilities, modalities, parameters, pricing, source links, and verification metadata.
- OpenAPI documentation served through `/openapi.json`, Swagger UI, and Scalar.
- Docker image build support for running the API service.
- Catalog validation tooling for schema, alias, lifecycle, freshness, pricing, parameter, and source checks.
